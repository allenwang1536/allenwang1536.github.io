MedX A/B Testing

Data Analysis

HTML/CSS/Excel | Spring 2024

Overview

Introduction

The task was to improve the effectivenss of MEDx's appointment manage page by making small visual modifications. An A/B test on users was conducted in order to evaluate the efficacy of these changes and decide whether we should implement them or not.

Proposed Modifications

Designs

The left side is the original version (version A), and the right side side is the modified version (version B).

User Task

The task that I assigned the user was to schedule an appointment with Adam Ng, MD at Morristown Medical Center on April 23, 2024

Modified Features

  • Change text color to black
  • Sort appointments by date descending
  • Restructure appointmennt information in order of doctor name, group, then type.

    • Hypotheses

    Misclick Rate (the frequency with which the users click something else on the page before finding the correct button for the task):

    Null Hypothesis: Version B has the same misclick rate as Version A

    Alternative Hypothesis: Version B has a lower misclick rate than Version A since the black colored text makes things easier to read.

    Prediction: I predict that we will reject the null hypothesis because changing the text color to black makes it easier to read the words and make users less prone to clicking the wrong button.
    Time on Page (the time spent on the webpage for each user group before completing the task):

    Null Hypothesis: Version B has the same time on page as Version A

    Alternative Hypothesis: Version B has a lower time on page than Version A since the new structure of the appointment information makes it easier for users to navigate and find their desired appointment successfully.

    Prediction: I predict that we will reject the null hypothesis because the appointments are sorted in ascending date order, and the relevant information about the doctor and group are highlighted more prominently, making it easier to navigate to the desired appointment quicker.
    Time to First Click (the amount of time that elapses before users make a first click):

    Null Hypothesis: Version B has the same time to first click as Version A

    Alternative Hypothesis: Version B has a lower time to first click than Version A since the black colored text and structured ordering of appointment times by date will make it more intuitive for users to find their desired appointment.

    Prediction: I predict that we will reject the null hypothesis because the sorted order of dates will allow users to navigate to the appointment more quickly.

    Data Collection

    Process

    Over the course of an hour, I had 19 users test out both versions A and B of the website on 2 different computers, tracking a number of different statistics about them.

    • Data Analysis

    Misclick Rate

    Test: I conducted a Chi-Squared Homogeniety test for this metric since we are trying to determine if there is a significant difference in the distribution of did_mislick (which is a categorical variable). After conducting the test, we get the following results:

  • df: 1
  • chi^2: 0.6333333333
  • p-value 0.4261352551

    • Analyis: A chi-squared statistic of 0.6333 with 1 degree of freedom does not indicate a large discrepancy between the expected and observed frequencies of misclicks in versions A and B of the website. Furthermore, the p-value of 0.4261 is much greater than 0.05, which suggests that the difference in misclick rates between the two website versions is not statistically significant.

    Conclusion: Since the p-value is greater than the standard alpha level (0.05), we fail to reject the null hypothesis. The null hypothesis in this context would be that there is no difference in the misclick rates between the two website versions. Therefore, based on this test, we would conclude that there is no evidence of a significant difference in the misclick rates for versions A and B of the website.

    Time On Page

    Test: I conducted a One-Tailed T Test since we are trying to determine whether there is significant difference in a certain direction (time spent on page for B is lower than A). After conducting the test, we get the following results:

  • Avg(A): 23826.68421
  • Variance(A): 261924913.9
  • Avg(B): 9561.368421
  • Variance(B): 20137519.91
  • Degrees of Freedom: 20.75151657
  • T-score (B - A): -3.702419
  • P-value: 0.0006707193483

    • Analyis: The p-value of 0.0006707193483, or about 0.067%, is far below 0.05. This indicates that the probability of observing a difference as large or larger than the one observed is very small if there were actually no difference in the population means (null hypothesis). The negative T-score of -3.702419 indicates that the average time on page for version B is significantly lower than for version A. The degrees of freedom are sufficient for the t-distribution to closely approximate the normal distribution, as it is greater than the standard 20 degrees of freedom.

    Conclusion: Given the very low p-value in this updated data, we can reject the null hypothesis for the one-tailed test. This suggests that the difference in the time on page between versions A and B is statistically significant, with version A having a significantly longer time on page than version B.

    Time to First Click

    Test: I conducted a One-Tailed T Test since we are trying to determine whether there is significant difference in a certain direction (time to first click for B is lower than A). After conducting the test, we get the following results:

  • Avg(A): 12945.05263
  • Variance(A): 46435300.39
  • Avg(B): 5302.894737
  • Variance(B): 4325521.433
  • Degrees of Freedom: 21.32460841
  • T-score (B - A): -4.675504556
  • P-value: 0.00006238110132

    • Analyis: The p-value of approximately 0.000062, or 0.0062%, is much lower than the standard alpha level of 0.05. This indicates that there is a statistically significant difference in the time on page between versions A and B, with version A having a longer time to first click. The negative T-score of -4.6755 indicates that the average time to first click for version B is significantly lower than for version A. The degrees of freedom are sufficient for the t-distribution to closely approximate the normal distribution, as it is greater than the standard 20 degrees of freedom.

    Conclusion: Given the very low p-value in this updated data, we can reject the null hypothesis for the one-tailed test. This suggests that the difference in the time to first click between versions A and B is statistically significant, with version A having a significantly longer time to first click.

    Conclusion

    We were able to reject the null hypotheses for time on page and time to first click. However, we failed to reject the null hypothesis for mislick rate. Therefore, the modifications we made to version B have been shown to be partially successful in improving user experience. To increase certainty that version B is an improvement, we should conduct more tests with additional data and metrics.
